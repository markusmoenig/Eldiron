# Todo List for v1

## Text Adventure Support

Would be great to have a text adventure tool switch, when enabled the dock would not show the tile-picker but a textareaedit to enter descriptions for the current sector. Linedef selections could enter the description / name of connections between the connected sections.

Items placement, entity placement could all be the same. Eldiron could than support full text based adventures.

Todos for this

- Make a client trait to hide the wgpu dependencies for clients which do not use a graphics interface (terminal). This is a bigger job.
- Implement command parsing for text adventure based games (optionally via AI).

## Better and more configurable Screen Widgets

Current screen widgets are aimed at in-game widgets, like buttons and text. But for signup, character setup we need more UI based widgets.

- Text lists
- Icon lists
- Text entry fields
- Drop down lists

Probably we will also need a general DnD system for items (drop spell from spell books into buton widget etc).

Also widgets for in game story-telling:

- Showing images
- Scrolling text

## Timeline Mode

RPGs are all abou telling a story. We need a timeline mode where characer actions are not script based but based on events on a timeline.

These can be played live during games.

## Raytracer

For screenshots and marketing we need a raytrace / pathtrace offline mode in the editor.

- Explosions
