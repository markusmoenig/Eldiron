<Admonition type="note" title="Retired Workflow">
    The older procedural terrain workflow has been retired from the current 3D editing path.
</Admonition>

The current 3D workflow is based on direct **Geometry Object** editing: create boxes, edit faces/edges/vertices, draw surface guides, cut openings, add ridges or grooves, and apply tiles or colors directly to geometry.

See [Creating 3D Maps](creating_3d_maps) for the active 3D map authoring workflow.
